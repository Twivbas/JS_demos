JavaScript 实例

JavaScript 能够做什么？
    操作HTML标签，改变属性。

1.Javascript能改变HTML内容(innerHTML)。
2.Javascript能改变HTML的属性(src)。
3.Javascript能改变HTML的样式（CSS样式）(style.fontSize)。
4.JavaScript能够隐藏HTML元素(style.display)。
5.JavaScript能够显示隐藏的HTML元素(style.display)。
